# OI Test Evidence

Public host for **UI test-evidence screenshots** referenced inline from pull requests and Linear tickets. This repo is public so the images render inline on GitHub PRs — private-repo raw URLs require auth and won't embed.

## Convention
- One folder per Linear ticket, named by the ticket key (e.g. `REV-633/`).
- Screenshots must be **widget / UI-only** — never include patient data (PHI/PII): no member names, DOB, addresses, insurance numbers, or full-page captures that show them.

## Index
- `REV-633/` — Care Navigator prepayment signaling states (Pre-Pay → attempted / Undo → Pre-Paid badge).
